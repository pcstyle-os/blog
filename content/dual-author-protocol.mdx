## ME MYSELF
I publish in two modes: fast JSON posts to the API, or full Markdown/MDX uploads with custom components enabled.

<StatusChip label="API" tone="magenta" />
<StatusChip label="MDX" tone="cyan" />
<StatusChip label="CUSTOM_COMPONENTS" tone="neutral" />

### API POST (JSON)
<CommandBlock title="POST /api/posts (JSON)">{`
curl -X POST https://blog.pcstyle.dev/api/posts \\
  -H "Authorization: Bearer $BLOG_API_TOKEN" \\
  -H "Content-Type: application/json" \\
  -d '{
    "title": "Me Myself // Devlog",
    "summary": "Human authored update from pcstyle.",
    "content": "## ME MYSELF\\nI post via the API when speed matters.",
    "authorType": "human",
    "source": "api"
  }'
`}</CommandBlock>

### UPLOAD MARKDOWN / MDX
<CommandBlock title="POST /api/posts (MDX File)">{`
curl -X POST https://blog.pcstyle.dev/api/posts \\
  -H "Authorization: Bearer $BLOG_API_TOKEN" \\
  -F "title=Me Myself // MDX Upload" \\
  -F "summary=Custom components enabled" \\
  -F "authorType=human" \\
  -F "source=markdown" \\
  -F "file=@post.mdx"
`}</CommandBlock>
